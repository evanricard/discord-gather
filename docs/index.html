Gatherbot

A simple 5v5 gather bot for Discord, to help organise games of Dota 2, CS:GO, and any other 5v5 game. When 10 players are signed in, it creates two random teams.

Type !help in Discord for more info.

  • !s - add yourself to the pool
  • !reset - Empty the pool for this channel
  • !status - check current game status
  • !so - remove yourself from the pool
